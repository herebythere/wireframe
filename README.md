# Wireframe

`Wireframe` is a collection of radically impactful web components.

## Mission statement

Every component is:
- based on an [aria pattern](https://www.w3.org/WAI/ARIA/apg/patterns/)
- rtl compatible
- theme-able
- implemented with CSS where possible
- a web component if not
